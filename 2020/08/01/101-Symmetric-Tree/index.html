101. Symmetric Tree

发表于 2020-08-01 分类于 LeetCode

如何判断一个树是不是对称？

  1. 如果一个树是空的，那它是对称的。
  2. 判断左子树和右子树是否对称。

如何判断左子树和右子树是否对称？

  1. 如果左子树和右子树都为空，那它们是对称的。
  2. 如果左子树和右子树只有一个为空，那它们不是对称的。
  3. 判断左子树和右子树的根节点的值是否相等。
  4. 判断左子树的左子树和右子树的右子树是否对称。
  5. 判断左子树的右子树和右子树的左子树是否对称。
